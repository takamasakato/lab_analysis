# Odor_response Analysis

in vivoカルシウムイメージングデータに対して、
前処理から細胞応答の定量評価、空間mappingまでを一貫して行った解析例です。

---

## 解析フロー
- ΔF/F算出（前処理・正規化）
- 刺激応答の時系列解析
- Glass’s Δを用いた刺激特異的細胞応答の定量評価
- 応答細胞の空間mapping（heatmap）

## 実装内容
- in vivoカルシウムイメージングデータの前処理パイプライン構築（ΔF/F算出・ベースライン補正）
- 刺激時刻に基づく時系列応答解析アルゴリズムの実装
- 各ROIの刺激応答量の自動定量化（peak/AUCベース）
- Glass’s Δを用いた刺激特異的細胞応答の効果量評価
- ROI位置情報と応答量を統合した空間マッピング処理の実装
- 閾値判定および再現性条件を考慮した応答細胞の分類ロジック設計

---

## 研究との関連
本解析は、自身の研究で扱うin vivoカルシウムイメージングデータの解析基盤として開発したものです。
嗅覚刺激に対する小脳プルキンエ細胞の神経活動を1細胞レベルで定量評価し、  
時系列応答と空間分布を統合的に解析することを目的として実装しました。

実験データの特性（ノイズ・個体差・試行間変動）を考慮し、  
単純な平均比較ではなく効果量（Glass’s Δ）を用いた評価指標を導入することで、  
刺激特異的な神経応答の客観的判定を可能にしました。  
研究における神経活動解析・図作成・定量評価までの一連の解析工程に実際に使用しています。
