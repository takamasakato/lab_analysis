# lab_analysis

大学研究におけるカルシウムイメージング解析および神経活動解析の例をまとめています。
（研究守秘義務に基づき、許可されたコードおよび再構成した解析例のみ公開）

## 内容
・CaImAnを用いたカルシウムイメージング解析例  
（動き補正、細胞抽出、時系列信号解析）

・匂い刺激応答におけるプルキンエ細胞の時空間応答マッピング解析例  
（刺激依存的ROI応答の時系列・空間統合解析）

・小脳プルキンエ細胞およびユーリデンドロイド細胞の比較解析例  

## 使用技術
Python / CaImAn / NumPy / pandas / matplotlib  
1細胞レベルの神経活動データに対する前処理〜抽出〜統計解析までの一連の解析フローを実装
