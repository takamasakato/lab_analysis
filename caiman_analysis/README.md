# caiman_analysis

in vivoカルシウムイメージングデータを対象とした、  
1細胞レベル神経活動解析（CaImAn）の実装例です。  
（研究守秘義務に基づき、再構成した解析例のみ公開）

本コードでは、実験データの前処理から細胞抽出、  
時系列信号解析までを一貫した解析パイプラインとして実装しています。

---

## 解析フロー
1. 動き補正（Motion Correction）
2. ノイズ低減・前処理
3. 細胞抽出（CNMF / CaImAn）
4. ΔF/F算出（正規化）
5. 時系列神経活動解析
6. 刺激応答の定量評価

---

## 解析結果例
・1細胞レベルでの神経活動時系列の抽出  
・ROIごとの刺激応答の定量評価

## 使用技術
- Python  
- CaImAn  
- NumPy  
- pandas  
- matplotlib  

大規模カルシウムイメージングデータに対し、  
自動化された細胞抽出と時系列解析を実装しています。

---

## 実装内容
・CaImAnによる動き補正と細胞抽出の自動化  
・ΔF/F計算および時系列信号処理  
・ROIごとの神経活動データ抽出  
・刺激タイミングに基づく応答解析  
・再現性を意識した前処理条件の統一設計  

手動解析によるばらつきが課題であったため、  
解析手順の標準化と自動化に重点を置いて実装しました。

---

## 研究との関連
本解析は、共焦点レーザー顕微鏡を用いて撮影された、プルキンエ細胞の1細胞レベル神経活動解析の一部として実装したものです。  
実験データ特性（動き・ノイズ・個体差）を考慮し、  
前処理から解析までを一体として設計しています。
